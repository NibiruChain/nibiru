# Nibiru EVM (Nibiru V2)

{{ $frontmatter.description }}

## Is Nibiru EVM live on mainnet?

Yes. **Nibiru EVM** (often called the **Nibiru V2** upgrade) is live on Nibiru mainnet
(`cataclysm-1`). The chain runs EVM and Wasm in parallel: you can deploy Solidity
contracts, use familiar Ethereum tooling, and interact with Nibiru-specific features
such as precompiles and FunTokens.

For an overview of what Nibiru EVM includes, see [Nibiru EVM](../../evm/index.md).
For developer setup, see the [Developer Hub — EVM guides](../../dev/evm/index.md).

## How do I connect a wallet or RPC?

Use standard EVM wallets (for example MetaMask) with the Nibiru EVM RPC and chain
ID listed on [Nibiru Networks and RPCs](../../dev/networks/index.md) and in the
[EVM FAQ](../../dev/evm/faq.md).

Mainnet defaults:

- RPC: `https://evm-rpc.nibiru.fi`
- Chain ID: `6900` (`0x1AF4`)

Block explorer: [nibiscan.io](https://nibiscan.io)

## Where can I read about the launch timeline?

[June 2024 alpha testing release](../../evm/news/evm-alpha-test.md) and the

For broader protocol and ecosystem updates, see
[Ecosystem Updates](../../ecosystem/updates/index.md).

## Are there user incentives or campaigns?

Incentive programs and partner campaigns change over time. Check
[Ecosystem Updates](../../ecosystem/updates/index.md) and official Nibiru community
channels for current announcements. This FAQ page only describes EVM availability
on the chain, not active promotions.
