# Liquid Staked NIBI (stNIBI)

> *[User Guides](../index.md) > [Tokens of Nibiru](./index.md) > [Liquid Staked NIBI (stNIBI)](./stnibi.md)*

{{ $frontmatter.description }}

Liquid staking allows you to put NIBI to work and earn rewards automatically while retaining the ability to trade or use stNIBI in blockchain applications.
