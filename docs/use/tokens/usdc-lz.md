## How to Acquire

- **Nibiru Bridge App:**
  Use the [Nibiru Bridge](https://app.nibiru.fi/bridge) to transfer USDC.e between Nibiru and other supported EVM chains.

- **Stargate Bridge App:**
  Bridge directly using [Stargate Finance](https://stargate.finance/bridge?srcChain=ethereum&srctoken=0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48&dstChain=nibiru&dstToken=0x0829F361A05D993d5CEb035cA6DF3446b060970b).
