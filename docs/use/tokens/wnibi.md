# Wrapped NIBI (WNIBI)

> *[User Guides](../index.md) > [Tokens of Nibiru](./index.md) > [Wrapped NIBI (WNIBI)](./wnibi.md)*

{{ $frontmatter.description }}

WNIBI exists to make NIBI available as an ERC-20 token within the Nibiru EVM.
Users can wrap native NIBI into WNIBI (and unwrap back) to interact with EVM
contracts, liquidity pools, and dApps on Nibiru. This ensures broad compatibility
across applications that require ERC-20 tokens.

## Metadata

| Token | Wrapped NIBI (WNIBI)   |
| --- | --- |
| Symbol | WNIBI | 
| Type |  ERC-20 token on Nibiru EVM, LayerZero OFT |
| Decimals | 18 |

## How to Get WNIBI

- **Wrap/Unwrap on Nibiru:**
  Use the [Nibiru Web App](https://app.nibiru.fi/) to convert between native NIBI and WNIBI.
  Wrapping converts NIBI into its ERC-20 equivalent, while unwrapping converts back to native NIBI.

- **Bridge to Other Chains:**
  WNIBI can be bridged across supported networks (e.g., Ethereum and Base) using the [Nibiru Bridge](https://app.nibiru.fi/bridge).
